# Tic Tac Toe Game in React
This project is a simple implementation of the classic Tic Tac Toe game built using React. The game allows two players to take turns marking spaces in a 3Ã—3 grid, with the goal of placing three of their marks in a horizontal, vertical, or diagonal row.

## Features
- Two-player gameplay (player X and player O)
- Dynamic game board updates
- Winning logic to determine the game result
- Simple and intuitive user interface

## Technologies Used
- React
- JavaScript
- CSS
